# Security Policy

## Reporting Security Issues

The Chaosblade maintainers take security seriously. We appreciate your efforts to responsibly disclose your findings.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them through our [private vulnerability reporting](https://github.com/chaosblade-io/chaosblade-io/security/advisories/new) form.

For more details, see the [Chaosblade security policy](https://github.com/chaosblade-io/chaosblade-io/blob/main/SECURITY.md).
